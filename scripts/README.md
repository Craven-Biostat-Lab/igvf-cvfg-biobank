# Scripts

This folder hosts vignettes and analysis reproduction scripts:

- `scalable-vus` contains scripts for reproducing the *All of Us* biobank validation presented in [**A scalable approach to resolving variants of uncertain significance**](https://doi.org/10.64898/2026.02.14.705848).
- `replicate-barrett` contains a vignette for replicating the analysis in Figures 1 and 2 from Barrett et al. on *All of Us* data.
